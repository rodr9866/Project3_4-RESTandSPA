Interesting Findings

  • The number of murders in St. Paul decreased from 2019 to 2020. In 2019, there was 10 and in 2020 there was 3.
  • In 2020 the neighborhood with the most crimes was North End. The neighborhood with the least crimes was Summit Hill.
  • The number of robberies in the Macalester-Groveland neighborhood increased from 2019 to 2020. In 2019, there were only 9, and in 2020 there are 25.
  • The neighborhood of West 7th has more total arson reports (27) than the Summit Hill neighborhood (6).

Tools Used

  • Vue.js
    • A framework for JavaScript used to build web interfaces and one-page applications.
  • Leaflet
    • API that allows you to render interactive maps in your web page as well as overlay data on top of the map. The map can be modified either by user interaction or by JavaScript commands.
  • Nominatim
    • API that allows you to search for locations by name, or retrieve a location's name based on lat/long coordinates.
  • St. Paul Crime
    • API that we created. Obtains crime records from database based on specified inputs.
